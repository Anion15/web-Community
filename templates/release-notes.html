🗓️ 2025년 4월 30일

  • 보안 코드 추가
  • 세션 인증, IP 인증, UUID 및 MAC 주소 기반 인증 등 다양한 방식 도입

🗓️ 2025년 5월 1일

  • 📖 "상정인사이드란?" 소개 서비스 시작

🗓️ 2025년 5월 2일

  • 📊 상정인사이드 역사 서비스 시작
  • 총 데이터 전송량 측정 기능 포함 서버 코드 추가

🗓️ 2025년 5월 3일

  • 모든 서비스 하단에 Footer 추가
  • 모바일 환경 호환성 향상 (너비 511px 이하 대응)
  • Footer 상단에 게시물 갯수를 표시하는 기능 추가

🗓️ 2025년 6월 7일

  • 후원시스템 'Buy me a coffee'으로 변경
  • 모든 서비스 우클릭 디자인 변경
  • 홈 화면 헤더 우측 버튼을 화면 크기에 맞춰 반응형으로 조정

🗓️ 2025년 6월 24일

  • request.remote_addr를 get_real_ip()로 변경하여 (cloudflare tunnel에서 127.0.0.1로 표시되는 문제 해결)
  • 광고차단기 감지 로직 추가
  • 광고 z-index; 조절하여 footer에 가리지 않도록 수정

🗓️ 2025년 6월 25일

  • VPN차단 로직 추가
  • 서버 로그 저장 형식 변경
  • 크롬 엣지 사파리 외의 브라우저 차단 페이지 추가

🗓️ 2025년 7월 4일

  • 모바일 사용자를 위한 햄버거메뉴 업데이트
  • 설정 기능 추가
  • 다크모드 추가
  • 모든 데이터 통신 규칙 수정 업데이트

🗓️ 2025년 7월 5일

  • cloudflare turnstile에서 새로 인증하거나 재인증시 높이 변경 고정
  • Footer에 개인정보 처리방침 바로가기 표기

🗓️ 2025년 7월 6일

  • 모든 푸터 영역에 '개인정보처리방침' 버튼을 시각적으로 강조하여 배치함.
  • 모든 페이지에 다크/라이트 모드 및 우클릭 디자인 설정(활성화/비활성화)이 반영되었습니다.
  • 처음 방문하는 사용자에게 쿠키 사용 및 개인정보처리방침에 대한 안내를 제공합니다.
  • 모바일 가로 폭이 좁은 환경에서 헤더와 main div들이 화면 밖으로 넘쳐보였던 현상을 수정하였습니다.
  • 처음 방문하는 사용자가에 상정인사이드 공식 디스코드 가입에 대한 안내를 제공합니다.

🗓️ 2025년 7월 9일

  • 서비스 이용약관 추가
  • 모든 페이지에서 30초 또는 60초의 캐시를 적용하였습니다.
  • 링크 하이라이터 추가
  • 상정인사이드가 아닌 다른 사이트로 이동하는 url을 클릭하였을 때 경고알림 추가
  • '인터넷 연결이 끊어졌습니다'알림 추가

🗓️ 2025년 7월 11일

  • 모든 알림 메시지 디자인 변경 (모든 alert 메시지의 디자인 추가)
  • 모든 페이지에서 30초 또는 60초의 캐시를 적용하였습니다.
  • 링크 하이라이터 추가
  • 상정인사이드가 아닌 다른 사이트로 이동하는 url을 클릭하였을 때 경고알림 추가
  • 모든 팝업창 다크모드 적용
  • 글자수 제한 표시 수정 (글 작성의 제목 입력부분과 내용 입력 부분의 우측 하단에 글자수 제한을 시각적으로 보여줌)

🗓️ 2025년 7월 12일

  • '이번주의 핫토픽'추가 (이번주동안 달렸던 게시물중에 반응이 가장 뜨거웠던 게시물을 선정하여 글 작성 아래에 고정으로 표시)
  • 핫토픽 css 애니메이션 부드럽게 수정
  • 최적화 코드 및 여러 버그 패치 완료
  • 댓글 가독성 패치

🗓️ 2025년 7월 13일

  • 구버전 브라우저 감지 페이지 추가 (구버전 브라우저를 통해 상정인사이드에 접속할 경우 디자인이 깨지는 현상을 발견, 사용자가 구버전 브라우저를 통해 접속을 시도할 경우 최신 브라우저를 통해 접속하도록 안내)
  • 모든 페이지에서 30초 또는 60초의 캐시를 적용하였습니다.
  • 핫토픽 선정 개선 (추천/비추천 중심 (40% 가중치), 댓글 활동 (30% 가중치), 참여도 비율 (20% 가중치), 시간 가중치 (10% 가중치))
  • 개발자 페이지 업데이트 (게시물 작성자 수정, 게시물 내용 수정, 게시물 삭제, 게시물 추천비추천 수정 기능 추가, IP 밴 기능 추가)

🗓️ 2025년 7월 25일

  • 비속어 검열기능 추가 (베타버전)

🗓️ 2025년 8월 3일
